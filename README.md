# Claude 自製 Skills

這些是為「文字探勘與行銷應用」課程製作的 Claude Skills。

## 📦 包含的 Skills

| Skill | 說明 | 觸發關鍵字 |
|-------|------|-----------|
| resume-health-check | 履歷健診（資管系大三生申請實習） | 幫我看履歷、履歷健診 |

## 🚀 如何使用

1. 在 Claude.ai 左側選單點「Skills」
2. 點「Add Skill」
3. 貼上 SKILL.md 的 GitHub Raw 連結：
   例如：https://raw.githubusercontent.com/tumitu/claude-skills/main/resume-health-check/SKILL.md
4. 儲存後即可使用
