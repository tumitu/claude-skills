# Claude 自製 Skills

這些是為「文字探勘與行銷應用」課程製作的 Claude Skills。

## 📦 包含的 Skills

| Skill | 說明 | 觸發關鍵字 |
|-------|------|-----------|
| resume-health-check | 履歷健診（資管系大三生申請實習） | 幫我看履歷、履歷健診 |

## 🔧 如何安裝此 Skill

**Step 1：下載 SKILL.md**

點以下連結，右上角點「Raw」，
對頁面按右鍵 → 另存新檔 → 存成 SKILL.md：

https://github.com/tumitu/text-mining-course/blob/main/resume-health-check/SKILL.md

**Step 2：上傳到 Claude.ai**

1. 前往 claude.ai
2. 左側選單 → Skills
3. Create skill → Upload a skill
4. 選擇下載的 SKILL.md
5. 儲存完成

**Step 3：測試**

在對話框輸入「幫我看履歷」即可啟動
